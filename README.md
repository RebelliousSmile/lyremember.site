# lyremember.site

Site web pour présenter l'application Lyremember - une application pour mémoriser vos découvertes musicales.

## Description

Ce site est construit avec Nuxt 3 et présente l'application Lyremember, ses fonctionnalités et ses cas d'utilisation.

Lyremember vous aide à capturer et organiser vos souvenirs musicaux. Ne perdez plus jamais une chanson que vous avez aimée.

## Installation

Installer les dépendances :

```bash
npm install
```

## Développement

Lancer le serveur de développement sur `http://localhost:3000` :

```bash
npm run dev
```

## Production

Construire l'application pour la production :

```bash
npm run build
```

Prévisualiser localement la build de production :

```bash
npm run preview
```

Ou générer un site statique :

```bash
npm run generate
```

## Structure du projet

- `/pages` - Pages de l'application (index, features, about)
- `/layouts` - Layout par défaut avec navigation
- `/public` - Fichiers statiques
- `/app.vue` - Point d'entrée de l'application

## Technologies

- **Nuxt 3** - Framework Vue.js pour applications web
- **Vue 3** - Framework JavaScript progressif
- **TypeScript** - Superset typé de JavaScript
